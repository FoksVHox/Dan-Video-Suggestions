# Video Suggestions
Want to see a video on DanFMN's YouTube? Suggest them [here](https://github.com/FoksVHox/Video-Suggestions/issues)!

## Suggesting a video
Please make sure that you:
- Avoid tags on your issues, we have the labels for a reason:
    - [Tag] < suggestion >
    - Tag: < suggestion >
- Make sure that there isn't a open issue with your suggestion already
